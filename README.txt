1.Se recomienda utilizar el editor de codigo "Visual Studio Code" para una mejor visualizacion del codigo y 
ejecucion del programa.

2. El primer paso para observar de manera correcta la evidencia es tener instalado en el computador MySQL Workbench,
 el cual una vez antes de instalar pide crear una contraseña para el usuario root, dicha contraseña se debe cambiar 
 por la que trae el archivo "key.js" en donde se establecen los datos para la conexion a la base de datos.

3. Para crear la base de datos en la carpeta "database" se encuentra la sintaxis o el codigo para crear la base de
 datos, el cual se puede poner directamente en MySQL workbench o en la consola de mysql "MySQL Command Line Client"

4. Se debe abrir una terminal integrada dentro de Visual Studio Code, la cual automaticamente aparecera posisionada 
sobre la ubicacion de la carpeta del programa que contiene los archivos; una vez dentro de ella escribir el comando 
"npm run dev" el cual permite que se ejecute el programa, e indiicara en la terminal la conexion establecida con la 
base de datos y en el local host.

5. Una vez concretada la conexion, en el navegador de preferencia escribir en el buscador "localhost:4000" el cual 
es el puerto en el que se encuentra el programa.

6. El primer usuario que se registre será considerado como administrador y todos los demás después de este serán 
considerados como usuarios clientes.
